## 🇧🇷 Sobre o Projeto

Jogo da velha desenvolvido com JavaScript puro, com foco em lógica de programação e manipulação de DOM.  
Permite que dois jogadores joguem alternadamente até vitória ou empate.

---

## 🇺🇸 About the Project

A Tic Tac Toe game built with vanilla JavaScript, focused on logic implementation and DOM manipulation.  
Allows two players to play alternately until a win or a draw.
